## Why this file exists

`SOUL.md` is the agent's personality, isolated from everything else. Editing this file changes how every reply sounds, immediately, with no code changes. ~500 tokens. The leverage is dramatic.

The persona should be specific and consistent. Generic personas ("be helpful and friendly") get washed out by the rest of the prompt. Sharp, distinctive personas hold up.
